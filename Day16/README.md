# What is a Repository?

A repository (repo) is a location used to store and organize files and folders related to a project.

A repository can contain:

- Documents
- Images
- Source code
- Markdown files
- Notes
- Videos
- Other project files

---

# Local Repository

A Local Repository is a repository stored on your own computer.

Examples:

- Files on your Desktop
- Notes stored on your laptop
- Practice labs saved locally

---

# Remote Repository

A Remote Repository is a repository hosted on a remote service such as GitHub.

Examples:

- GitHub repositories
- Shared project repositories
- Online code storage platforms

Organizing files and folders helps to:

- Track learning progress

- Manage projects efficiently
- Build a professional GitHub portfolio
